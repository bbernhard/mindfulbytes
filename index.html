mindfulbytes - an API to the past

Remember what happened today - x years ago.

Get it

Index your digital assets

Index all images in your local and remote (e.g Nextcloud) folders by EXIF date.

Features

Easy to extend with plugins

Fully dockerized

Powerful REST API

Built-in Signal Messenger & Email notifications

100% Open Source

Example

Get an image that was taken at this day, x years ago or return an random image if no image was found. 
curl -X GET http://127.0.0.1:8085/v1/topics/imgreader/images/today-or-random?caption=This%20image%20was%20created%20{{timeago}}
